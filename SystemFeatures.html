System Features


The Google search Engine has two important features that help it produce high precision results. They are:

1) Link Structure
2) Anchor text

1) Link structure

Link structure is one of the two system features of Google that helps in producing high precision result. Google make use of link structure of the web to calculate a quality ranking for each web page. This ranking is called PageRank. Google utilizes links to improve search results.

Anchor text Anchor text is treated in a special way in search engine. Most search engine associate the text of a link with the page that the link is on. This has several advantages. First, anchors often provide more accurate descriptions of web pages than the pages themselves. Second, anchors may exist for documents which cannot be indexed by a text-based search engine, such as images, programs and database. This makes it possible to return web pages which have not actually been crawled. The idea of propagating anchor text to the page it refer to was implemented in the World Wide Web Worm especially because it helps search non-text information. Anchor text propagation is mostly used because it can help to provide better quality results.
